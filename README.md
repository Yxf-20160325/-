# 聊天室

基于 Socket.IO 的实时多人聊天室，支持私聊、群聊、文件上传、实时音视频通话等功能。

## 功能特性

- 用户注册、登录、好友系统
- 多人房间聊天
- 私聊消息
- 文件、图片、语音消息发送
- 实时位置共享
- 视频通话 / 语音通话
- AI 聊天助手
- 管理员功能（公告、系统设置）
- 多种小游戏（贪吃蛇、俄罗斯方块、石头剪刀布）

## 技术栈

- 前端：HTML5 + CSS3 + JavaScript
- 后端：Node.js + Express + Socket.IO
- 实时通信：WebSocket
- 视频通话：WebRTC

## 运行

```bash
npm install
node server.cjs
```

访问 http://localhost:147
