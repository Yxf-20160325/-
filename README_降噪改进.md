# 聊天室降噪功能改进文档

## 改进概述

本次降噪功能改进显著提升了音频通话质量，增加了更智能、可配置的降噪功能，提供了更好的用户体验。

## 主要改进点

### 1. 先进的音频处理算法
- **实时噪声估计**：新增实时噪声估计算法，能够动态检测和建模环境噪声
- **语音概率检测**：基于能量、过零率、峰峭因子等多维度特征计算语音概率
- **自适应降噪**：根据噪声水平和语音概率自动调整降噪强度
- **频率选择性降噪**：保护重要的语音频段（1-4kHz），同时抑制环境噪音

### 2. 增强的音频处理链
- **多段均衡器**：
  - 高频增强滤波器（2500Hz）：增强语音清晰度
  - 低频削减滤波器（80Hz）：减少环境嗡嗡声
  - 中频增强滤波器（1000Hz）：提升语音区域的音质
- **动态范围压缩器**：平滑音量波动，避免爆音
- **自适应噪声门**：智能检测声音并过滤背景噪声

### 3. 用户友好的控制界面
- **三级降噪强度**：
  - 低强度：保留更多原音，适合音乐传输
  - 中强度：平衡降噪与音质，默认选项
  - 高强度：强力降噪，适合嘈杂环境
- **高级音频设置**（可展开）：
  - 噪声门阈值控制（-60dB至-20dB）
  - 语音清晰度增强控制
  - 自适应降噪开关
  - 语音保护模式开关

### 4. 实时噪声监测
- **通话界面指示器**：实时显示噪声水平和语音状态
  - 噪声水平条：可视化显示噪声强度
  - 语音检测指示灯：识别语音活动
  - 颜色编码：绿色为良好，黄色为中等，红色为警告
- **设置面板监测**：高级设置中显示实时噪声数据

### 5. 智能交互功能
- **降噪按钮**：根据降噪级别显示不同颜色
  - 绿色：低强度降噪
  - 黄色：中强度降噪
  - 红色：高强度降噪
- **一键切换**：点击即可开启/关闭降噪功能
- **状态同步**：降噪状态在所有界面实时同步

## 技术实现细节

### 算法改进
1. **谱减法增强**：基于实时噪声估计的谱减法降噪
2. **软拐点处理**：避免硬切换，提供平滑的降噪过渡
3. **多维度特征提取**：能量、过零率、方差、峰峭因子等特征
4. **频率加权**：优先保护语音重要的频段

### 用户体验优化
- **视觉反馈**：所有控制都有即时的视觉反馈
- **详细提示**：每个设置都有详细的解释说明
- **渐进式披露**：基本设置在前，高级设置可展开
- **实时更新**：所有设置变更都立即应用到通话中

### 性能考虑
- **低延迟处理**：使用高效的ScriptProcessor节点
- **资源优化**：仅在通话中激活噪声监测
- **自动清理**：通话结束时自动释放资源
- **兼容性**：降级支持使用浏览器原生降噪功能

## 使用方法

### 快速使用
1. 进入聊天室并加入房间
2. 发起或接听音频/视频通话
3. 点击通话界面中的降噪按钮（🔇）开启降噪
4. 通过按钮颜色判断当前降噪强度

### 高级设置
1. 点击通话界面中的设置按钮（⚙️）
2. 在设置面板中切换到"通话设置"标签
3. 展开"高级音频设置"部分
4. 根据需要调整各个参数

### 噪声监测
1. 开启降噪后，噪声监测指示器会自动显示
2. 观察噪声水平条了解当前环境噪声
3. 查看语音状态指示灯了解语音检测情况
4. 点击指示器可刷新显示数据

## 期望效果

1. **安静环境**：保持语音自然，仅抑制微小杂音
2. **办公室环境**：有效过滤键盘声、白噪声
3. **嘈杂环境**：强力抑制背景噪音，保持语音清晰
4. **音乐传输**：低强度降噪模式保留音乐音质

## 兼容性说明

- 支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
- 降级支持：如果Web Audio API不可用，自动使用浏览器原生降噪
- 移动端优化：界面和控制针对移动设备进行优化
- 跨设备同步：设置会在同一浏览器中保持

## 故障排查

### 常见问题
1. **降噪效果不明显**：尝试提高降噪强度级别
2. **语音被过度抑制**：开启"语音保护模式"
3. **音量波动大**：开启"自动增益控制"
4. **按钮无响应**：确保通话已建立且音频流正常

### 调试建议
1. 检查浏览器控制台是否有错误信息
2. 确保麦克风权限已授予
3. 尝试不同的音频设备
4. 重启浏览器或清除缓存
