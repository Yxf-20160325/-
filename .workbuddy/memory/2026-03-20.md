## Render 部署 Socket.IO 问题修复

### 问题描述
- **本地环境**: `localhost:147` 可以正常加入房间
- **Render 部署**: `https://liaotianshi.onrender.com/` 无法加入房间

### 根本原因
根据 Render 官方文档分析,问题可能出在:
1. Socket.IO 客户端配置未针对 Render 平台优化
2. WebSocket 在 Render 负载均衡器下的连接问题
3. 传输方式(transports)配置需要调整

### 修复内容

#### 客户端优化 (`public/index.html`)
1. **Socket.IO 配置优化**:
   - 添加明确的 `path: '/socket.io/'` 配置
   - 根据协议动态选择传输方式:
     - HTTPS 环境优先 WebSocket: `['websocket', 'polling']`
     - HTTP 环境优先 polling: `['polling', 'websocket']`
   - 统一心跳配置: `pingInterval: 25000`, `pingTimeout: 60000`
   - 添加 `upgrade: true` 和 `rememberUpgrade: true`

2. **添加调试监听器**:
   - `connect_error`: 记录连接错误详情(消息、描述、上下文、当前传输方式)
   - `connect_timeout`: 记录连接超时
   - 增强现有 `disconnect` 和 `reconnect` 监听器

#### 服务器端优化 (`server.cjs`)
1. **Socket.IO 配置优化**:
   - CORS 添加 `allowedHeaders: ["Content-Type", "Authorization"]`
   - 添加 `allowUpgrades: true` 和 `allowEIO3: true`
   - 添加 `connectTimeout: 45000`
   - 统一心跳配置与客户端一致

2. **添加调试日志**:
   - 记录用户连接信息 (socket.id, IP, 传输方式)
   - 监听传输方式升级事件 (polling → websocket)
   - 记录 User-Agent 信息

## 额外修复: socket.on() 在未初始化时调用的错误

### 问题描述
```
(索引):6980 Uncaught TypeError: Cannot read properties of undefined (reading 'on')
    at (索引):6980:16
```

### 根本原因
第 6980 行的 `socket.on('opponent-left', ...)` 直接在全局作用域中执行,而不是在 `initSocket()` 函数内。这意味着当页面加载时,`socket` 变量还没有被赋值,导致 `Cannot read properties of undefined` 错误。

### 修复内容
1. 将 `socket.on('opponent-left', ...)` 包装到 `registerOpponentLeftListener()` 函数中
2. 在 `initSocket()` 的 `connect` 事件中调用 `registerOpponentLeftListener()`
3. 添加 `socket.off('opponent-left')` 移除旧监听器,避免重复注册

### 修复后的代码
```javascript
// 注册对手离开事件监听器
function registerOpponentLeftListener() {
    // 移除旧的监听器(如果存在)
    socket.off('opponent-left');
    
    // 监听对手离开事件
    socket.on('opponent-left', (data) => {
        console.log('对手离开了游戏:', data);
        showNotification('对手已退出游戏', 'warning');

        // 关闭打字对战模态框
        if (data.gameType === 'typing') {
            closeTypingGame();
        }
        // 关闭俄罗斯方块模态框
        if (data.gameType === 'tetris') {
            closeTetrisGame();
        }
    });
}
```

在 `initSocket()` 的 `connect` 事件中:
```javascript
socket.on('connect', () => {
    // ... 其他监听器注册
    
    // 在连接成功后注册对手离开事件监听器
    if (typeof registerOpponentLeftListener === 'function') {
        registerOpponentLeftListener();
    }
    
    // ... 其他监听器注册
});
```
