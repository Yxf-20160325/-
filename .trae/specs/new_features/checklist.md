# 聊天室新功能 - 验证清单

## 消息引用回复功能
- [ ] 点击消息的引用按钮后，消息输入框显示引用的原始消息
- [ ] 发送回复后，聊天界面显示引用的原始消息和回复内容
- [ ] 引用消息显示原始消息的发送者和时间
- [ ] 回复消息显示在引用消息下方，形成清晰的对话脉络
- [ ] 引用回复在私聊中能正常显示
- [ ] 引用回复在群聊中能正常显示

## 深色模式
- [ ] 设置界面中添加了深色模式切换开关
- [ ] 切换深色模式开关后，界面立即变为深色
- [ ] 深色模式下界面背景和文字颜色适应夜间使用
- [ ] 所有界面元素在深色模式下都能清晰显示
- [ ] 刷新页面后，深色模式设置保持不变
- [ ] 重新打开浏览器后，深色模式设置保持不变

## 消息转发功能
- [ ] 消息操作区域添加了转发按钮
- [ ] 点击转发按钮后，显示转发目标选择界面
- [ ] 选择目标并确认后，目标用户收到转发的消息
- [ ] 转发消息显示原始发送者信息
- [ ] 支持选择多条消息进行转发
- [ ] 转发消息在私聊中能正常显示
- [ ] 转发消息在群聊中能正常显示

## 聊天背景设置
- [ ] 设置界面中添加了聊天背景设置选项
- [ ] 提供了默认背景选项
- [ ] 支持上传自定义背景图片
- [ ] 选择默认背景后，聊天界面背景立即更新
- [ ] 上传自定义背景图片后，聊天界面背景更新为自定义图片
- [ ] 刷新页面后，背景设置保持不变
- [ ] 重新打开浏览器后，背景设置保持不变
